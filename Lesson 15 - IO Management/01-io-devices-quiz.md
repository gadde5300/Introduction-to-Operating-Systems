# Quiz 1: I/O Devices

For each **device**, indicate whether it's typically used for **input (I)**, **output (O)**, or **both (B)**.

- Keyboard
- Microphone
- Speaker
- Network interface card (NIC)
- Display
- Flash card
- Hard disk drive

A: The answers are as follows:

- Keyboard: **I**
- Microphone: **I**
- Speaker: **O**
- Network interface card (NIC): **B**
- Display: **O**
- Flash card: **B**
- Hard disk drive: **B**
