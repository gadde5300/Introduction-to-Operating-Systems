# Quiz 4: Bare-metal or Hosted

Do you think that the following **virtualization products** are **bare-metal/hypervisor-based (HV) or host-OS-based (OS)**

- KVM
- VMware ESX
- Fusion
- Citrix XenServer
- VirtualBox
- Microsoft Hyper-V
- VMware Player

A: The answers are as follows:

- KVM: OS
- VMware ESX: HV
- Fusion: OS
- Citrix XenServer: HV
- VirtualBox: OS
- Microsoft Hyper-V: HV
- VMware Player: OS
